# Blender Scripts

The Blender scripts are grouped by responsibility to keep the modeling stage readable and reusable.

## Subdirectories

- `animation/`: dataset generation, gesture animation, and random-pose synthesis.
- `core/`: shared rig configuration, reusable gesture code, and helper utilities.
- `export/`: frame-export and script-export helpers.
- `reconstruction/`: JSON-driven motion reconstruction utilities.
